# RUN'23

A mini-game coded in two evenings for the RUN'23 ("Reuni√≥n de Usuarios de MSX del Norte") held in Hoznayo (Cantabria, Spain) on April 15th, 2023.


## Summary

Four players. One MSX. The last one standing wins.

## Gameplay

Just play it. You'll get it very quickly.

## Controls

- Player #1: `F1`, `F2`
- Player #2: `Z`, `X`
- Player #3: `N`, `M`
- Player #4: `Ins`, `Del`
